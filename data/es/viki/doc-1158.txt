Sitio web
Un sitio web o un sitio Internet es un conjunto de páginas web relacionadas entre sí y puestas en línea en una dirección web (URL).
En 2006, la web superó el número de 100 millones de sitios.
Las páginas web están escritas en un lenguaje, a menudo HTML o PHP, pero también CSS, Javascript, Ajax, scripts de flash, etc.
Estas páginas se envían a un servidor, un computador que permanece encendido las 24 h del día y al que nos conectamos para acceder al sitio web, y cada vez que un visitante solicita visitar una página, le es enviada transformada en lenguaje PHP, que es un lenguaje servidor, o en lenguaje HTML para que el navegador Web del visitante pueda leer correctamente la página (un navegador es un software que permite ver sitios web).
Los dos sitios más visitados son Google y Facebook
Para acceder a un sitio web se tiene que escribir una dirección electrónica (URL) en la barra de direcciones del navegador.
Este te llevará automáticamente al sitio Web, excepto si se usa un buscador, entonces se escribe el nombre de lo que quieres encontrar y el buscador te muestra varios sitios web relacionados con la búsqueda.
Existen varios tipos de sitios web, con distintos tipos de permisos al crear textos o añadir imágenes, como los wikis.
servicios de comercio electrónico: Amazon, eBay, Kelkoo...
servicios de correo electrónico: Outlook, Gmail, Yahoo, sin registro...
La web no es la misma cosa que Internet.
Internet existía antes de la invención de la web.
El conjunto de sitios web no es Internet.
El correo electrónico (e-mail) y la descarga de archivos (FTP), los grupos de noticias (usenet), el lanzamiento del software de control remoto (telnet), la búsqueda de documentos (Gopher) son las funciones de Internet que existían antes de la web.
Para hacer tu sitio web, cosa que puede parecer compleja, puedes ayudarte de sitios web que te explicarán cómo hacerlo y de múltiple software.
