Lenguaje de programación
Un lenguaje de programación es un código que se utiliza para crear un programa.
Este código está formado por:
un conjunto de órdenes
unas reglas de sintaxis (es decir, una forma correcta de usar esas órdenes).
Para crear un programa es necesario conocer las órdenes y la sintaxis del lenguaje de programación elegido.
Algunos lenguajes de programación usados en la escuela son el logo y el basic.
