Dirección IP
En la imagen vemos que cada computadora tiene una IP diferente.
Una dirección IP es una dirección de Internet.
El nombre deriva del protocolo utilizado para realizar el direccionamiento y el enrutamiento: Internet Protocol.
Esto quiere decir que cada ordenador que se conecta a Internet debe tener un número que le identifique dentro de esta red para poder pedir y recibir información.
Como si fuera la dirección de correos de una casa en una ciudad, pero sin nombres de calles ni ciudades ni países, solo mediante un número.
Una dirección IP se representa por cuatro grupos de números separador por puntos, por ejemplo: 123.456.78.9.
Esta dirección es única, no se puede repetir.
La dirección IP puede ser fija o variable (dinámica).
Es suministrada por la empresa que proporciona el servicio de conexión a Internet.
Si se tiene contratada una IP fija, siempre que el ordenador se conecte a Internet se tendrá la misma dirección, pero si se tiene contratada una IP dinámica ésta puede variar cada vez que nos conectemos.
