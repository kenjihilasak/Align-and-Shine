Logotipo de Windows actual (2021-presente)
Windows es el nombre de una familia de sistemas operativos creados por la empresa Microsoft desde 1985.
Hay varias versiones de este sistema operativo en idiomas muy diferentes (incluido el español).
La interfaz gráfica predeterminada utiliza CShell desde Windows 10 .
El sistema operativo no es gratuito, pues la empresa que lo hace, Microsoft, lo desarrolla con ánimo de lucro, sin embargo, el sistema es descargado de Internet varias veces ilegalmente.
Creado en 1985, es el primer sistema operativo de Windows con interfaz gráfica monousuario de 16 bits.
El anuncio original mostraba un costo de $99 dólares americanos
Fue creado en 1990, fue mejorado con un microprocesador modelo 80386.
Ahora es un competidor para Apple Macintosh.
Junto a ella aparecieron las versiones 3.1 y NT
Creado en 1995, fue el primer sistema operativo multitarea preemptiva, como IBM OS/2, pero tenía competencia con dicho sistema operativo por su interfaz gráfica
Creado en 1998,
incluía nuevos controladores de hardware y el sistema de ficheros FAT32 (también soportado por Windows 95 OSR 2 y OSR 2.5)
En 2000, Microsoft introdujo Windows ME, significado de Millenium Edition (Edición del Milenio) que era una copia de Windows 98 mejorada y más rápida, con más aplicaciones añadidas.
Windows ME fue un proyecto rápido de un año para rellenar el hueco entre Windows 98 y el nuevo Windows XP, y eso se notó mucho en la poca estabilidad de esta versión.
En ese mismo año dio la luz Windows 2000, nombre en reemplazo de la versión de Windows NT 5.0, nueva y muy útil para los administradores de sistemas y con una gran cantidad de servicios de red y lo más importante: admitía dispositivos Plug&Play que venían siendo un problema con Windows NT.
En 2001, con la unión de las familias Windows NT y Windows 9.x se creó el Windows XP en su versión Home Edition y Professional Edition.
Windows XP usa el núcleo de Windows NT 5.1 e incorpora una nueva interfaz, y hace alarde de mayores capacidades multimedia.
Windows XP ha sido muy popular debido que en las redes sociales hacen memes con los sonidos de dicho sistema.
¿Qué onda con el XP?
Windows XP utiliza estas siglas con respecto a las letras que contiene la palabra Experience, ya que estas siglas garantizaban una experiencia máxima al utilizar el sistema operativo.

Antes del desarrollo del Vista, en 2005, Windows desarrolló un sistema operativo que no se lanzó oficialmente, era el Longhorn, que solo era una versión mejorada de Windows XP para enfocarse al próximo proyecto, que es el Vista.
Se lanzó solo la versión para servidores que eran solo para pruebas de desarrollo.
El Windows Longhorn hace referencia al zodíaco Tauro ¡Caramba!
Apareció en el mercado el 30 de enero de 2007 en el núcleo Windows NT 6.0.
Cabe destacar los continuos retrasos en las fechas de entrega del sistema operativo.
Inicialmente se anunció su salida al mercado a inicios-mediados de 2006; posteriormente y debido a problemas durante el proceso de desarrollo, se retrasó su salida hasta finales de 2006.
El último retraso trasladó la fecha hasta finales de enero de 2007.
Estos continuos retrasos han llevado a Microsoft a tomar diversas medidas para minimizar los gastos extras derivados de los retrasos.
Lo más destacable con esta versión, es que tenía muy poca compatibilidad con diversos programas, lo cual hacía este sistema el peor Windows del 2007.
Esto hizo que muchas personas siguieran usando Windows XP, a pesar de ser muy antiguo para la época.
Es el sucesor de Windows Vista, de núcleo Windows NT 6.1 y fue planeada para un desarrollo de tres años como plazo.​ Anteriormente se la conocía por los nombres código de Blackcomb y Vienna.
Fue lanzado el 22 de octubre de 2009.
Consiste en un sistema operativo similar a su predecesor, Windows Vista, pero en este caso, era mucho más estable y más confiable que Vista, además de ser muy ligero y de requerir muy pocos recursos de computadora, algo que Microsoft obtuviera mucho éxito con sus productos, por ello, es hasta ahora, uno de los sistemas operativos más usados, a pesar de que su soporte terminó en enero de 2020.
Logotipo de Windows 8 y 8.1
Es el sucesor de Windows 7, fue lanzado en el 2012, y se diseñó de tal forma que todas las plataformas de Microsoft puedan funcionar en él (PC o computadoras personales, teléfonos celulares, tabletas electrónicas o tabletas y las consolas Xbox).
Su diseño fue diseñado mayormente para ser utilizado en tabletas, pero no para computadoras de pantalla sin interacción táctil, lo cual causó un gran descontento entre usuarios y esto hizo que se obtuvieran errores de diseño como la eliminación del Menú Inicio.
Es el sucesor de Windows 8, en él, se arreglaron apresuradamente algunos errores de su antecesor, trayendo el Menú Inicio y más personalización para el usuario.
Muchos consideraron que no habían muchas diferencias a su antecesor, debido a que habían bastantes características que todavía aquejaban a muchos usuarios.
Fue presentado el 30 de septiembre de 2014 en San Francisco e incluye el regreso del menú de Inicio, un sistema de tareas completamente renovado y cambios en el símbolo del sistema, entre otros.
Está desarrollado con el Núcleo Windows NT 6x y unirá definitivamente a Windows Phone, Windows RT y Windows en un único sistema.
Además, contarán todos los dispositivos con una única Tienda Windows.
Windows 10 tiene la capacidad de conectarse con una Xbox One gracias a la compartida interfaz entre ambos.
Además de ser actualmente uno de los sistemas operativos más usados en la actualidad.
Es el nuevo sistema operativo de Microsoft, Presentado en Junio de 2021 como una prueba de Insider, y es lanzado al mercado el 4 de Octubre de 2021, como el proyecto hacia la productividad empresarial.
Aunque esté basado en la arquitectura de Windows 10, Windows 11 ha dado un cambio radical en su aspecto, como la barra inicial, que ahora está al centro; También la nueva característica TPM, en la que ya se puede usarse hasta 7 ventanas a la vez y hacer las tareas tanto cotidianas como empresariales, sin perder nada de memoria.
