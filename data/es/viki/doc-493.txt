Microsoft Edge
Microsoft Edge (o simplemente Edge) es un navegador web desarrollado por Microsoft, actualmente basado en el proyecto de código abiertoChromium.
Fue lanzado por primera vez para Windows 10 y Xbox One en el 2015, como el navegador predeterminado de éstos, para los sistemas móviles Android e iOS en el 2017 , y para MacOS en 2019.
Fue diseñado para reemplazar a Internet Explorer, el navegador web que venía incluido en Windows desde 1995, ya que éste ha quedado obsoleto con el paso del tiempo y sólo servía para visualizar páginas antiguas.
Microsoft Edge inicialmente estaba diseñado para ser usado únicamente en Windows 10, con un motor de renderizado llamado EdgeHTML5, único para el navegador, algo que garantizaría que fuese más rápido, y la implementación de la clásica e, que sería una versión moderna del antiguo logo de Internet Explorer, pero a muchos usuarios de Windows no les gustó, debido a que ésta les recordaba a éste navegador, algo que Microsoft arregló lanzando una nueva versión basada en el proyecto de código abierto Chromium, dándole una nueva apariencia, resultando un navegador web ligero, potente y fácil de usar, según Microsoft, lanzándolo oficialmente el 15 de Enero de 2020, para los sistemas operativos Windows 7, 8, 8.1 y 10, al igual que para Android y MacOS.
