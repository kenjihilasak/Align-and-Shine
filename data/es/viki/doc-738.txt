El alfabeto griego es el alfabeto usado por el idioma griego desde el siglo 9 a.e.c.
Los griegos usaron el alfabeto fenicio como base para crear el alfabeto griego, que fue el primero en tener letras diferentes para todos sus sonidos, tanto vocales como consonantes.
El alfabeto griego es también la base de los alfabetos etrusco, latino, y cirílico.
Aunque el alfabeto sigue siendo usado hoy para escribir el griego moderno, ha cambiado en algunos aspectos: por ejemplo, la pronunciación de cada letra ha cambiado desde el siglo 5 a.e.
C. También el griego antiguo usaba varios diacríticos para modificar los sonidos de algunas letras, a este sistema se le llamaba ortografía politónica, pero hace poco tiempo, la ortografía se simplificó para usar solo dos signos: el acento grave y la diéresis.
