Windows 8.1
Pantalla de Inicio de Windows 8.1
Windows 8.1 es una versión del sistema operativo Windows.
Fue lanzado el 18 de octubre del 2013.
Esta versión consistía en una actualización gratuita al sistema operativo de Windows 8, su predecesor.
Esta versión al principio sólo estaba disponible para usuarios activos de Windows 8 en la llamada "Tienda de Windows" (o Windows Store en inglés), aunque más tarde, Microsoft anunció que daría copias gratuitas en su sitio web oficial.
Las diferencias entre Windows 8 y 8.1 eran mayor compatibilidad, más velocidad y más facilidad de manejo.
Sin embargo, continuaban las quejas y problemas de manejo con respecto a Windows 8 por parte de una parte de usuarios, para los cuales Windows seguía dando "dolores de cabeza", como por ejemplo, el mismo problema del Menú Inicio (que tuvo el esperado regreso del botón, que había sido eliminado en Windows 8).
Sin embargo, el Menú Inicio seguía siendo a pantalla completa si bien era más personalizable en esta versión que en Windows 8.
Ello obligó a Microsoft a sacar la versión actual de Windows 10, ya que este incluía el regreso del Menú Inicio y había solucionado más problemas de Windows 8.
