Dark
web
Attention
, ne pas confondre avec le deep web.
Le dark web ("web obscure" en anglais), souvent confondu avec le deep web ("web profond" en anglais) est un réseau qui utilise des protocoles spécifiques intégrant des fonctions d'anonymat.
Il existe en parallèle des réseaux privés, des Darknet.
Il ont Les même spécificités que le Dark web mais sont un peu moins utiliser pour des ventes d’armes, de drogues.
Il servent à créer, en quelque sorte, un réseaux social d’échanges et de discutes totalement privé et inaccessible.
Le dark web n'est pas accessible avec des navigateurs classique (Chrome, Firefox, Edge, etc...) mais avec le logiciel Tor.
Les URL du dark web se terminent tous par ".onion".
Sur le dark web il est fortement conseillé d'utiliser un VPN, pour cacher son adresse IP.
Ce qu'on y trouve peut être très dangereux que ce soit pour vous ou pour votre entourage :
Les virus sont très fréquents, c'est pour ça qu'en accédant à ce deep web sans être protégé d'un pare-feu et d'un antivirus on peut se faire pirater son ordinateur très rapidement.
Mais pas seulement, dans certains pays où la censure est appliquée par le gouvernement (par exemple la Corée du Nord) l'utilisation du dark web est le seul moyen pour les citoyens de pouvoir exprimer leurs opinions sans devoir craindre pour leur vie.
Les transferts d'argent qui sont effectués sur cette partie d'internet sont règlés en Bitcoins, une monnaie qui à l'avantage d'être intraçable.
Le dark web est aussi utilisé par des lanceurs d'alertes qui veulent se protéger du gouvernement.
