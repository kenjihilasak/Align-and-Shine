Dart (langage)
Sommaire
Dart un langage de programmation orientée objet : on y retrouve les notions de classe, d'interface, d'héritage, etc.
Sa syntaxe est dérivée du langage C, le code est entouré des accolades.
Avec Dart, le programmeur n'est pas obligé de préciser le type d'une variable : on dit Dart est un langage à typage faible.
