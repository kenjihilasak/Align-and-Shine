Erreur 404 sur la Wikipédia allemande, 2012
En informatique, une erreur 404 est un code d'erreur HTTP transmis par le serveur concerné pour indiquer que l'URL n'existe pas.
Le message type est généralement 404 Not found, en français «
Non trouvé », mais en fonction du serveur et de son logiciel (Apache, Nginx), le message par défaut change.
Cependant, le fondateur du site peut personnaliser cela et créer des pages d'erreur originales1.
Certains sites Web renvoient les internautes à une page consacrée à l'erreur2.
On peut aussi détourner ce code erreur de son utilisation première et le renvoyer lorsqu'un visiteur demande une page avec restriction d'accès, par exemple à cause d'une censure.
Le visiteur a inséré une URL incorrecte.
Le visiteur a suivi un lien rompu (détissage de la Toile).
