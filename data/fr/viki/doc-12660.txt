Myers Briggs Type Indicator
Les quatre axes du MBTI.
Le Myers Briggs Type Indicator, plus souvent désigné MBTI, est un test de personnalité proposé par Isabel Briggs Myers et Katherine
Cook Briggs en 1962
qui fonctionne selon quatre préférences ;
l'une pour la circulation de l'énergie (E pour extraversion ou I pour introversion), une autre pour la collecte des informations (S pour la sensation, N pour l'intuition), une troisième pour la prise de décision (T pour la pensée dont le verbe anglais est to think, F pour le sentiment dont le verbe anglais est to feel) et une quatrième pour le mode de passage à l'action (J pour le jugement, P pour la perception).
Ces quatre axes étant chacun constitués de deux catégories, il existe selon ce modèle un total de seize types de personnalités que l'on écrit en rassemblant les quatre lettres caractéristiques des catégories (exemples : ISTJ, ENFJ, ESTP, INFP, …).
Le test ne fonctionne cependant pas selon ces axes mais essaie de déterminer dans quel ordre sont les fonctions principales et auxiliaires de chaque individu.
Ainsi, une seule lettre de différence entre deux types donne des personnalités différentes.
L'objectif de ce test n'est pas de classer les personnalités, mais de comprendre comment d'autres personnes peuvent fonctionner.
Ainsi, il peut être utile de savoir que certaines personnes recherchent l'harmonie quand d'autres recherchent la vérité, certaines se basent sur les faits pour juger une situation pendant que d'autres retiennent l'intention des personnes, certains pensent qu'il ne faut jamais remettre au lendemain ce qui peut être fait maintenant alors que d'autres pensent qu'il ne faut jamais faire maintenant ce qui aura probablement changé demain.
Le MBTI permet de se donner des axes d'amélioration de soi-même dans une optique de développement personnel et peut servir d'aide à la compréhension des autres ainsi qu'à la façon dont ils fonctionnent.
Dans le monde du travail, il peut être utilisé pour former des groupes plus productifs et qui fonctionnent mieux.
À ce titre, il n'existe pas de bon ou de mauvais type de personnalité mais seize types avec des qualités et des défauts différents.
