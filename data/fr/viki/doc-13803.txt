Point
virgule
Le point-virgule est un signe de ponctuation.
C'est une virgule surmontée d'un point (;).
En général, le point-virgule peut se substituer à une virgule.
Il indique une pause plus longue.
Le point-virgule est utilisé pour séparer des propositions juxtaposées :
J'avais décidé de me rendre à la bibliothèque municipale, pour rendre une bande dessinée ;
manque de chance, elle était fermée ce jour-là.
Le point-virgule est aussi utilisé dans les listes à la fin de chaque item.
Le point-virgule est de moins en moins utilisé.
Dans une phrase écrite, le point-virgule est séparé par un intervalle aussi bien du mot qu'il suit que de mot qu'il précède.
Le modèle est : [
mot ;
mot ]
