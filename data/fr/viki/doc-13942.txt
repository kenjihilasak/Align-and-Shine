Nucléotide
Tableau comparatif de la composition des acides nucléiques
Un nucléotide est une molécule organique complexe dont l'hydrolyse fournit au moins une molécule d'ose (sucre).
Il est composé de trois parties :
une base nucléique variable
un sucre à cinq atomes de carbone,
un groupement phosphate identique
pour les nucléotides de l'ADN et de l'ARN.
Ce sont les éléments de base d'un acide nucléique.
Il existe à l'état naturel deux grandes catégories d'acides nucléiques, qui ont trois bases azotées communes :
Il existe 5 bases nucléiques classiques :
A pour adénine
C pour cytosine
Schéma de l'interaction entre bases nucléiques dans une double hélice d'ADN
G pour guanine
T pour thymine (présent seulement dans l'ADN)
U pour uracile (présent seulement dans l'ARN)
Deux nucléotides peuvent s'associer.
Cette association se fait selon des interactions spécifiques : C avec G et A avec T ou U, ce sont les interactions de Watson-Crick.
Ces interactions permettent notamment le décodage du code génétique ou la structure en hélice double brins de l'ADN.
Structure moléculaire de l'ATP.
Les P sont les trois phosphates, le pentagone est le sucre (ribose), la partie en haut à droite et la base azotée (adénine).
L'ATP est un nucléotide qui permet de stocker de l'énergie dans la cellule.
