Atom, processeur utilisé dans les objets connectés tels que les décodeurs, téléphones, etc...
Celeron, processeur utilisé dans les ordinateurs portables entrée de gamme, moins puissant que les Pentium
Pentium, processeur destiné à la bureautique, moins puissant que le Core i3
Core i3, entrée de gamme des modèles Core, utilisé pour la bureautique
Core i5, milieu de gamme des modèles Core, utilisé dans les jeux principalement
Core i7, haut de gamme des modèles Core, utilisé dans la retouche photo, les jeux extrêmement lourd, etc...
Core i9, très haut de gamme des modèles Core, utilisé dans la retouche photo, l'édition vidéo, les jeux lourds, la 3D extrême, etc...
Xeon, destiné aux professionnels, utilisé dans les serveurs et les stations de travail
