Windows Internet Explorer (anciennement Microsoft Internet
Explorer avant la version 7)
est un navigateur Internet et un logiciel propriétaire.
Il est livré avec Windows, le système d'exploitation le plus utilisé, ce qui en fait le navigateur le plus utilisé.
C'est d'ailleurs pour abus de position dominante que Opera Software (la société qui développe Opera) a porté plainte contre Microsoft : Opera Software trouve qu'étant donné que Windows est le système d'exploitation le plus utilisé, les utilisateurs n'ont pas le choix du navigateur.
La dernière version d'Internet Explorer est la version 11 disponible sur Windows RT et Windows 8.1 et Windows 7.
Les problèmes de sécurité ont été critiqués surtout pour la version 6 qui a été déconseillé de télécharger pendant quelques temps en raisons de ces problèmes de sécurité trop importants.
Les hackers recherchent aussi plus activement des vulnérabilités (appelées aussi failles).
La sécurité d'Internet Explorer a été renforcée.
Depuis 2015 avec la sortie de Windows 10, Internet Explorer est remplacé par Microsoft Edge, le nouveau navigateur internet de Microsoft.
Depuis la sortie de Microsoft Edge, Internet Explorer n'est plus mis à jour.
La dernière version d'IE est la version 11.
Certains sites web (même des sites internet très connus) ont décidé de ne plus supporter le navigateur Internet Explorer, à cause des nombreuses incompatibilités et à cause de la sécurité du navigateur.
