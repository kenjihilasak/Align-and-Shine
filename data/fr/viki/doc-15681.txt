Page Web
Une page
Web est une ressource du Web conçue pour être consultée par des visiteurs à l'aide d'un navigateur.
Elle a une adresse universelle.
Techniquement, une page Web est souvent constituée d'un document et d'images.
Cependant, tout type de ressources ou d'assemblage de ressources, textuelles, visuelles, sonores, logicielles, peuvent constituer une page Web.
