Table
Une table (et des chaises) dans une cuisine contemporaine
Une table est un meuble.
Généralement, elle est composée d'une planche plane soutenue par des pieds.
Sommaire
Il existe des tables de toutes les formes (carrée, rectangulaire, ronde, ovale...).
La taille et la hauteur de la table sont prévues pour son usage.
Par exemple, il existe des tables basses pour mettre devant un canapé, des tables hautes pour siège de bar, des tables-gigogne (des tables de tailles progressives, les plus petites entrent sous les grandes), des tables à langer (pour nettoyer les bébés)....
Le style de la table va souvent avec les chaises qui l'entourent.
Guéridon :
Petite table ronde à un seul pied.
console : table appuyée sur un mur ayant des pieds en forme de volute.
autel : table sur laquelle un prêtre célèbre des sacrifices religieux.
pupitre : table où s'installent les élèves pour travailler à l'école.
étal :
table où le boucher prépare la viande.
Ce peut être aussi la table sur laquelle un commerçant présente ses produits (on dit aussi étalage).
tables gigognes : ensemble de tables de différentes tailles qui peuvent se placer les unes sous les autres pour gagner de la place de rangement.
Le mot table peut aussi désigner l'ensemble des sujets traités dans un livre, avec le renvoi à la page concernée (c'est la table des matières).
