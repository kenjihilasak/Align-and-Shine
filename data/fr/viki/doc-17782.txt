Code
Un code peut être :
un ensemble de règles de comportement et de conventions, souvent non écrites, par exemple un code vestimentaire est la façon dont il est recommandé de s'habiller selon les situations, et ce que ça représente, un code de conduite ;
un document (livre ou autre) où des règles ou lois sont rassemblées (code juridique) :
un code d'accès ou un code secret est une sorte de mot de passe
