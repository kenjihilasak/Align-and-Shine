Écran bleu de la mort
BSOD sur Windows
L'écran bleu de la mort (BSoD, Blue Screen of Death) est le nom du message d'erreur affiché sur les ordinateurs Microsoft Windows mais on parle aussi d'écran vert de la mort (GSoD, Green Screen of Death) pour Windows 10 Insider, en cas d'erreur fatale.
BSOD sur Windows XP
Sur les systèmes d'exploitation Windows XP et Windows 2000, le BSOD survient lorsque le noyau du système d'exploitation a une erreur et qu'il ne peut pas la traiter.
Sur certaines versions beta de Windows Vista, Windows Longhorn on peut retrouver le Red Screen of Death (RSoD), qui est lui, rouge au lieu d'être bleu.
Le RSoD sur les versions beta de Windows Vista (Windows Longhorn)
Sur les versions de développement Insider de Windows 10, on peut trouver le Green Screen of Death (GSoD), qui est lui, est vert au lieu d'être bleu.
