LibreOffice est une suite bureautique libre et open-source, basée sur .
C'est une alternative à Microsoft Office.
Note : depuis sa version 6.2, LibreOffice a une interface à onglets (optionnelle) qui la rapproche visuellement de Microsoft Office.
Elle a été créée par The Document Foundation.
LibreOffice est écrit en C++, XML, Java.
La dernière version est la 6.4.7.
Une version 7.0.4 est en cours de développement.
Malheureusement, LibreOffice est peu utilisé face à son plus gros concurrent, Microsoft Office.
LibreOffice Math est un logiciel de formules mathématiques.
Il n'a pas d'équivalent Microsoft Office.
LibreOffice Base est un logiciel de gestion de bases de données, alternative à Microsoft Access.
Il se différencie cependant beaucoup de ce dernier par de nombreuses fonctionnalités soustraites ou ajoutées.
Une interface de LibreOffice pour les écoles est disponible chez PrimTux.
