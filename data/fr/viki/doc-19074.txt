Windows 7 (aussi appelé Windows Seven ou Win7) est un système d'exploitation qui est sorti le 22 octobre 2009.
Il succède à Windows Vista qui est souvent critiqué par sa lenteur d’exécution et par sa sécurité très faible.
Windows 7 est disponible dans les versions suivantes :
Au démarrage de Windows 7 (en anglais)
Édition Starter: utilisée sur les netbooks, elle demande peu de puissance et n'a pas beaucoup de fonctionnalités ;
Édition Familiale Premium: version vendue sur la plupart des ordinateurs portables.
Dans les pays en développement, elle s'appelle Familiale Basique et ne possède pas de la nouvelle l'interface Aero ;
Édition Intégrale: elle regroupe toutes les fonctionnalités des versions Familiale Premium et Professionnelle, plus quelques autres.
La dernière grosse mise à jour de Windows 7 est le SP1 (Service Pack 1) de 2011.
Le support de Windows 7 a pris fin le 14 janvier2020.
Depuis cette date, les mises à jour de Windows 7 sont payantes d'après Microsoft.
Windows 7 a les nouveautés suivantes : Nouvelle barre des tâches.
Windows 7 est aussi beaucoup plus rapide que son prédécesseur Windows Vista et beaucoup plus stable.
