Sommaire
Une des grandes nouveautés est une nouvelle interface graphique composée d'images interactives similaire à Windows Phone 8 ;
On peut aussi citer le démarrage très rapide de Windows.
L'objectif étant un démarrage en 8 secondes ;
Une prise en charge de certains équipements comme les écrans 3D ou les télévisions sans fil ;
Windows 8 est souvent critiqué pour son interface "Modern UI" appelé aussi "Metro" en raison de sa mauvaise intégration pour des ordinateurs non tactiles.
Il a été aussi critiqué à cause de la disparition du bouton/logo démarrer en bas à gauche (même si celui-ci a été remis dans Windows 8.1)
Windows 8.1 est sorti le 17 octobre 2013, c'est une mise à jour gratuite pour les utilisateurs de Windows 8 et de Windows RT, son téléchargement est disponible sur le Windows Store.
Le téléchargement de Windows 8.1 depuis Windows XP, Windows Vista ou Windows 7, ou autre version antérieure est payant et disponible seulement à partir du 18 octobre 2013.
Avant d'installer cette mise à jour, on peut la prévisualiser sur le Windows Store.
Le bouton Démarrer de la barre des tâches qui a été perdu sur Windows 8 est revenu avec de nouvelles fonctionnalités ;
le bouton Démarrer ne génère plus un menu.
Quand on clique gauche sur le bouton, on peut changer d'interface (Modern UI ou Metro pour passer vers le Bureau).
Quand on clique droite sur le bouton, on peut voir des interfaces spéciales comme pour accéder au panneau de configuration, par exemple.
La mise à jour fait passer Internet Explorer à la version 11.
Pour installer Windows 8.1 depuis Windows 8, tapez ceci dans la barre d'adresse de votre navigateur : ms-windows-store:PDP?PFN=. Vous êtes ensuite redirigés vers le Windows Store, cliquez sur Mettre à jour à Windows 8.1.
Vous pourrez prévisualiser Windows 8.1 avant la mise à jour.
Pour installer Windows 8.1 depuis une version antérieure à Windows 8, rendez-vous sur Microsoft Store ou Windows Store (site Web en ligne).
NB :
Pour installer Windows 8.1 depuis une version antérieure à Windows 8, il est conseillé de faire un reformatage complet du disque dur et de faire une réinstallation ou acheter un nouveau PC ou une tablette tactile par exemple Microsoft Surface.
Windows 8.1 Update 1 est une mise à jour de Windows 8.1, elle est sortie le 8 avril 2014, la mise à jour ajoute la barre de tâches dans l'écran d'accueil et dans les applications Metro et améliore l'utilisation de la souris et du clavier pour les PC non tactiles, cette mise à jour se fait depuis Windows Update12.
