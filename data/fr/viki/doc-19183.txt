Microsoft Paint est un logiciel de création et de modification d'image installé par défaut sur les PCWindows.
Il est installé par défaut depuis la première version de Windows (Windows 1.0) en 1985.
En juillet, Microsoft annonce la suppression de Paint pour Windows 10, les internautes se sont mobilisé pour le sauver, Microsoft a donc changer d'avis et Paint est disponible en téléchargement sur le Windows Store.
La nouvelle version de Paint est Paint 3D
qui figure sous le nom de Paint 2.
