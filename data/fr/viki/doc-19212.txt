Game Freak
Le projet commence sous le nom de Capsule Monsters en 1989.
Cependant, son projet est rejeté.
Le nom est changé pour le nom de Pocket Monsters.
De 1989 à 1996, le développement du projet a duré sept ans.
Trois ans après, le 27 février 1996 notamment, l'entreprise sort les deux premières versions de Pokémon Vert et Rouge sur Game Boy.
La version Bleu sort le 15 octobre 1996.
La version Jaune sort le 12 septembre 1998 avec comme Pokémon de départ, Pikachu qui reprend exactement les détails du dessin animé.
