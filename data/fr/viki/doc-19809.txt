Sommaire
Windows Update ou Microsoft Update est à la base un site Web pour installer des mises à jour (améliorations, correctifs de sécurité, ...) pour Windows et d'autres produits de Microsoft comme par exemple Microsoft Office ou Windows
Live Messenger (remplacé par Skype)
Les services packs sont téléchargeables depuis Windows Update.
Ce site Web est devenu un logiciel lors de la sortie de Windows 98.
L'installation de mises à jour nécessite une connexion Internet.
La vitesse de téléchargement et l'installation des mises à jour est variable selon la qualité et l'efficacité de la connexion Internet utilisée.
Les mises à jour automatiques ont pour principe d'installer les mises à jour importantes et recommandées automatiquement sans l'accord de l'utilisateur.
La plupart des mises à jour de Windows et d'autres produits Microsoft se font par le biais de Windows Update si les produits concernés sont installés sur l'ordinateur, bien entendu.
Pour accéder à Windows Update via Windows XP, cliquez sur le bouton Démarrer puis sur Tous les programmes et cliquez sur Windows Update, vous pouvez à présent regarder si votre ordinateur est à jour ou installer des mises à jour.
Pour accéder à Windows Update via Windows Vista ou Windows 7, cliquez sur le bouton Démarrer, tapez Update dans la zone de recherche et cliquez sur Windows Update, vous pouvez à présent regarder si votre ordinateur est à jour ou installer des mises à jour.
Pour accéder à Windows Update via Windows 8 ou Windows 8.1, allez sur le menu d'accueil (Modern UI ou Metro), tapez Update, cliquez sur Windows Update, vous pouvez à présent regarder si votre ordinateur est à jour ou installer des mises à jour.
