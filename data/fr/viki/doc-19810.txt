Windows XP
Logo de Windows XP
Windows XP (qui signifie Windows eXPerience
- aussi appelé WinXP) est un système d'exploitation qui est sorti le 25 octobre 2001, de la gamme Windows vendu par Microsoft.
Il est disponible en Édition Familiale (Édition Grand Public) et en Édition Professionnelle.
Il a succédé à Windows ME et Windows 2000.
Il précède Windows Vista.
Windows XP fait partie de la famille Windows NT.
Windows XP n'est plus supporté par Microsoft depuis le 8 avril 2014, depuis cette date, Microsoft a arrêté de donner des correctifs et des mises à jour de sécurité par Windows Update, ce qui rend les utilisateurs de Windows XP plus vulnérables aux virus.
Windows XP a commencé a être conçu en 1998.
A l'époque, Internet était moins présent.
De ce fait Windows XP n'a pas été conçu pour résister au virus, et malgré l'évolution de Windows XP, le code de Windows XP est toujours moins protégé des virus.
Le développement de Windows XP a commencé en 1999 avec le projet nommé Windows Neptune (Microsoft Neptune).
Sorti en 1999 en version alpha, Windows Neptune est basé sur les codes sources de Windows 2000.
Vers l'an 2000, les projets Windows Neptune et Windows Odyssey ont été fusionnés pour créés Whistler.
Windows Whistler est la version bêta de Windows XP.
