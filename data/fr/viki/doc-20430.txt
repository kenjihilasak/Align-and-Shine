Milieu de vie
Un milieu de vie (ou l'habitat), est l'endroit dans lequel vivent les êtres vivants.
Ils y trouvent de la nourriture, un support, un abri ...
Tout ce dont ils ont besoin.
Un être vivant peut avoir plusieurs milieux de vie :
La mousse pousse par terre, sur les arbres, sur les toitures ...
Les cloportes se trouvent sous les pierres, dans les souches d'arbres, sous les feuilles ...
Voici plusieurs milieux de vie différents :
Milieu d'une cour
Sous un buisson
Entre le mur et le chêne
Les êtres vivants n'ont pas les mêmes besoins :
Les cloportes ont besoin de moins de lumière, alors que les humains si.
