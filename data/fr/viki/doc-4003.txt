Code postal
Attention
à ne pas confondre !
Pour les sujets ou articles dits homonymes, voir : CP.
Un code postal (CP en abrégé) est un ensemble composé de chiffres, de lettres, ou des deux, qui permettent à la poste de faciliter la distribution du courrier (lettres, colis...).
Il est inscrit sur l'enveloppe du courrier ou du colis envoyé, au sein de l'adresse postale, et peut désigner une ville comme une aire plus grande.
Dans certains pays, le code postal est converti (« transformé ») en code-barrespostal pour améliorer le tri du courrier par des machines de traitement du courrier.
Sommaire
Il existe des différences entre les pays, que ce soit au niveau de la longueur du code postal, de son utilisation éventuelle de chiffres et de lettres, etc.
Le premier pays à avoir instauré le système de code postal est l'Allemagne, en 1941, suivie par le Royaume-Uni en 1959 et par les États-Unis en 1963.
Alors qu'en France et dans les pays européens en général, on inscrit le code postal avant le nom de la ville, dans la plupart des pays anglophones (qui parlent l'anglais), il est le dernier élément inscrit sur une enveloppe.
En France, le code postal a été mis en place en 1964 par La Poste et le format est passé à cinq chiffres à partir de 1972 : D'une manière générale, il s'agit, pour les deux premiers chiffres, du numéro (code) du département où sera envoyé le courrier ;
suivi de trois chiffres, qui correspondent au code de la ville.
Le plus souvent, les préfectures ont pour trois derniers chiffres le code 000.
Par exemple, la ville de Dijon, située dans le département de la Côte-d'or, a pour code postal 21000, 21 étant le code du département.
Certaines communes ont plus d'un code postal, notamment les grandes villes : Strasbourg a pour codes postaux, 67000, 67100 et 67200, répartis selon les quartiers.
D'autres villes ayant une population importante sont divisées en arrondissements municipaux, c'est le cas pour Paris (au nombre de 20), Marseille (16) et Lyon (9).
Dans ce cas, les derniers chiffres correspondent au numéro de l'arrondissement.
Paris a pour codes postaux les codes 75001 à 75020, avec une particularité pour le XVIe arrondissement, lui-même divisé en deux et comportant donc deux codes postaux : 75016 et 75116.
Marseille a pour codes postaux les codes 13001 à 13016.
Lyon a pour codes postaux les codes 69001 à 69009.
Bien qu'ayant respectivement le code de département 2A et 2B, la Corse-du-Sud et la Haute-Corse ont tous les deux comme deux premiers chiffres le nombre 20.
Dans de rares cas, certaines communes qui n'ont pas de bureau de distribution se font distribuer le courrier par le bureau d'un autre département.
C'est le cas, par exemple, de villages situés à la frontière d'un autre département.
Le village de Laveyrune en est un exemple : Bien que situé en Ardèche, il est situé à la frontière d'un département voisin, la Lozère et de ce fait, il utilise comme code postal le code 48250.
Au contraire des départements de France métropolitaine qui en ont 2, les départements d'outre-mer (DOM) et les collectivités d'outre-mer (COM) ont un code à 3 chiffres.
Mais tous les deux ont un code postal à 5 chiffres.
Dans ce dernier cas, les 2 derniers chiffres déterminent le bureau de distribution.
Par exemple, le code de la ville de Trois-Rivières en Guadeloupe est 97114 (le code de la Guadeloupe étant 971).
L'Armée française utilise un code postal dont les deux premiers chiffres sont 00.
Monaco, qui n'appartient pas à la France mais qui est un État indépendant, a adopté le système postal français.
Ainsi, Monaco a pour code postal 98000.
