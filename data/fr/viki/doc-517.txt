Code génétique
Le code génétique est contenu dans l'ADN.
L'ADN est comme un livre dans lequel se trouvent codés tous les caractères d'une espèce.
C'est ce qui fait que les êtres humains sont tous des êtres humains identiques, avec des cheveux, des dents, deux bras, deux yeux, etc. C'est le même code qui fait que les araignées sont des araignées et que les coquelicots sont des coquelicots.
La cellule doit savoir lire ce livre pour fabriquer les bonnes protéines, utiles à l'espèce concernée.
À gauche, un brin d'ARN messager avec sa séquence de bases azotées ;
à droite, les codons correspondants, chacun codant un acide aminé.
Les protéines sont codées par l'ADN.
Les protéines sont composées de l'enchaînement dans un ordre précis des 20 acides aminés.
Or, l'ADN ne comporte que 4 éléments : les bases azotées A, T, G et C.
Comment coder 20 acides aminés avec 4 bases ?
Un acide aminé correspond à une succession de trois bases azotées appelée triplet ou codon : par exemple, la séquence CCA correspond à l'acide aminé appelé proline, la séquence CGG correspond à l'arginine.
Il y a au total 64 combinaisons possibles, 3 séquences ne correspondent à aucun acide aminé et codent l'arrêt de l'assemblage des acides aminés (on les appelle codons stop).
Comme il reste 61 combinaisons pour 20 acides aminés, un acide aminé est codé en moyenne par 3 séquences différentes, qui donnent le même résultat (un changement dans la séquence ADN peut donc ne rien changer dans la protéine).
Si le code est CCACGG, alors la partie de la protéine sera proline-arginine, dans cet ordre.
Le code CGGCCA donne une autre combinaison, qui est arginine-proline.
Les protéines peuvent contenir des milliers d'acides aminés, le code sera donc très grand pour ces protéines.
Le code génétique est le même pour toutes les espèces de la Terre, à quelques exceptions près (il y a quelques organismes qui utilisent 1 ou 2 acides aminés en plus des 20 autres, il existe aussi quelques variantes dans le code génétique, un codon peut parfois coder un autre acide aminé que celui qu'il code normalement).
C'est pour cette raison qu'il est possible de faire produire à une fleur une protéine de bactérie par exemple.
On peut aussi déterminer qu'une certaine protéine rend une espèce résistante à un insecticide.
On peut utiliser ce gène et l'introduire dans une autre espèce pour qu'elle devienne à son tour résistante à cet insecticide.
Il s'agit alors d'un organisme génétiquement modifié ou OGM.
