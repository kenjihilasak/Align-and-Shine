Véhicule
Un véhicule est un objet utilisé pour transporter, avec ou sans matériel, déplacer une ou plusieurs personnes, sans que celles-ci ne touchent directement le sol.
Le but d'un véhicule est de faciliter le transport :
soit pour aller plus vite avec le même chargement ;
soit pour amener un chargement plus grand.
Par exemple, une péniche ne permet pas d'aller plus vite qu'à pied, mais permet d'emporter plus de matériel.
De même, une bicyclette permet d'aller plus vite, sans pour autant permettre de déplacer plus que son sac à dos.
Une façon de classifier les véhicules pourrait être par endroits où ils se déplacent.
On peut créer ainsi trois catégories principales:
les véhicules terrestres: ceux qui se déplacent sur la terre
les voitures
les motos
les vélos
les trottinettes
les skate-boards
les bus
les camions
les véhicules aériens: ceux qui se déplacent dans les airs ou qui volent
les avions
les montgolfières
les delta-planes
les planeurs
les hélicoptères
les véhicules aquatiques ou marins: ceux qui se déplacent dans l'eau ou sur l'eau
les bateaux
les sous-marins
les motos marines
