Sommaire
La première version date de 1985 (Windows 1.0).
Le premier qui a fait succès est Windows 3.1 les versions anciennes ayant fait un échec.
Il y a ensuite Windows 95, qui ajouté le menu démarrer et la barre des taches, dont le modèle graphique ressemble à celui du Macintosh.
Puis Windows 98, qui fut l'un des plus grand succès de Windows, il ajoute beaucoup plus de nouveautés par rapport à Windows 95.
Windows ME est un relatif échec à cause de sa grande instabilité.
Ensuite, Microsoft a décidé de ne plus faire qu'un seul type de logiciel d'exploitation : ce sera Windows XP, qui utilise des éléments de Windows 98 et Windows 2000.
Les particuliers et les entreprises avaient alors accès au même type de système, même s'il existe un version Pro et une version Familiale.
Windows Vista a tenté de succéder à Windows XP mais faisant aussi un grand échec à cause à sa grande instabilité, mais son successeur Windows 7 est sorti le 22octobre2009.
3 ans plus tard
en 2012, sort Windows 8 successeur de Windows 7, malheureusement Windows 8 fait un échec.
Une mise à jour de Windows 8, nommé Windows 8.1, est sortie en 2013, qui rencontreras un succès un peu mieux que Windows 8 .
Le logo de Windows est tout bleu depuis 2012.
Le logo de Windows est : le carré vert est en haut à droite, le carré rouge est en haut à gauche, le carré bleu est en bas à gauche, et le carré jaune est en bas à droite.
Microsoft Windows a réussit à imposer son système d'exploitation sur les ordinateurs de type PC.
Les concepteurs de logiciels pouvaient donc écrire des logiciels pour ce système sachant qu'un grand nombre de personnes pourraient être en mesure de l'utiliser.
C'est un point très important.
