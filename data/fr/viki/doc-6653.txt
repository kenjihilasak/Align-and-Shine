Complément du nom
Le complément de nom ou complément du nom (CDN, parfois aussi : complément déterminatif) est la fonction grammaticale d'un nom qui, dans une phrase, complète un autre nom.
En français, le plus souvent, le complément du nom est introduit par la préposition de.
Exemple : une bouteille d'eau : le nom eau, introduit par la préposition de, est complément du nom bouteille.
Autres exemples : la fille duroi, la ville demon premier amour, la bataille d'Austerlitz, l'ours bleu dePaul...
Parfois, les compléments de noms sont introduits par d'autres prépositions :
un livre enjaponais ;
une cuillère àsoupe ;
un coiffeur pourchauves ;
une compote avecpépins
Le rapport de sens entre le complément du nom et le nom complété est souvent :
la possession : la casquette demon frère ;
mais peut être aussi :
l'origine : du vin deBordeaux ;
la matière : un banc enpierre ;
le contenu : un verre debière ;
la destination : un verre àbière.
