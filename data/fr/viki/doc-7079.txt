Astérisque
Un astérisque est un caractère typographique qui ressemble à une étoile (*).
Attention
à ne pas le confondre avec l'astérisme (⁂) qui est formé par trois astérisques disposés en forme de triangle.
Sommaire
Ce symbole a plusieurs utilités dans des domaines différents.
En voici quelques-unes.
Lorsqu'il est placé après un mot, il signale une note en bas de page concernant ce mot.
On pourrait par exemple trouver dans un texte le mot «
astérisme* » et en bas de page : « * : L'astérisme est un symbole typographique formé par trois astérisques disposés en forme de triangle}}. Une seconde note pourra alors être suivie de deux astérisques (**), puis trois (***), puis quatre, etc ;
Dans un roman, quelques astérisques (souvent trois) à la suite signalent un changement de chapitre, ou de partie.
Pour ce faire, on peut aussi utiliser l'astérisme ;
Plusieurs astérisques peuvent aussi masquer quelques lettres d'un mot…
Soit trois astérisques,
soit autant qu'il n'y a de lettres masquées ;
Soit pour conserver l'anonymat : Mme T*** ;
Soit pour ne pas afficher une injure : P****n !
Les astérisques peuvent aussi être insérés
lorsque vous entrez un mot de passe, afin que personne ne puisse lire ce code derrière votre dos ;
Il peut également remplacer une étoile ;
Il peut également servir de tiret BBcode.
En programmation, il remplace souvent le signe de multiplication : 3x3=9 devient 3*3=9 ;
Sur un site Web, placé à côté d'un formulaire, il signale habituellement que ce champ est obligatoire ;
Il y a d'autres utilisations, mais trop techniques ou trop complexes pour être abordées ici.
Dans certains forums et chats de discussions, * fait une action, * signifie que la personne fait une action ;
Sur un téléphone, l'astérisque appelée «
touche étoile
» fait partie avec le # (dièse) l'une des deux touches spéciales.
C'est ce symbole typographique qui a inspiré le nom de Astérix le Gaulois.
D'ailleurs, certaines personnes font l'erreur de dire Astérix ou lieu d'astérisque par ce fait.
