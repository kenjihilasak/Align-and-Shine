Clé USB
Une clé USB ou clef USB est un périphérique informatique sur lequel on peut stocker des données (c'est donc une mémoire informatique), et qui, puisqu'il tient dans la poche, permet de transférer ces données d'un ordinateur à un autre.
Les clés USB servent à :
Transférer les données d'un ordinateur à un autre ;
Effectuer des sauvegardes si, par exemple, un pirate attaque l'ordinateur, les données sont toujours inscrites dans la clé USB.
Stocker des logiciels d'installation,
pour transformer la clé en installateur (pour mettre Windows ou Linux sur un ordinateur)
