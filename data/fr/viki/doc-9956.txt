Mode sans échec
Le mode sans échec est un mode de démarrage proposé par certains systèmes d'exploitation ou certains logiciels.
Il permet de démarrer avec le strict minimum : seul ce qui permet de démarrer le système est activé.
Ce mode est souvent appelé mode de diagnostic.
En effet, le système ne démarre que ce dont il a besoin, avec les paramètres par défaut.
Si un bug qui se produit habituellement ne se produit pas en mode sans échec, cela signifie qu'il provient d'un paramètre ou d'un logiciel qui fait planter le système.
Il est aussi utilisé pour supprimer des logiciels malveillants.
En effet, ledit logiciel n'étant pas démarré, on peut le supprimer sans qu'il puisse l'en empêcher.
L'accès au mode sans échec se fait, dans le cas des systèmes d'exploitation, soit dans les paramètres du système, soit, si le système refuse de démarrer complètement, par une combinaison de touches exécutée au démarrage de l'ordinateur.
Pour les logiciels, cela peut simplement être un clic droit sur l'icône, qui proposera un démarrage en mode sans échec.
