PNG
Le format PNG (Portable Network Graphics) est une extension de fichier, qui est utilisé pour les images.
Les fichiers étant de ce format se nomment "nom du fichier.png".
D'autres extensions existent comme le format GIF ou jpg.
Il est souvent utilisé lors de création d'image avec le logiciel Paint, qui le place par défaut en PNG.
